Instruções:

Instale o node.js
'npm install node'

instale o bootstrap
'npm install bootstrap'

instale o react
'npm install react'

-------------

Clone o repositório:


Entre na pasta ex00_projeto_api
Entre na pasta pokedex_api

Execute o comando:
'npm start'
Será automaticamente carregado no navegador pelo localhost:8080
